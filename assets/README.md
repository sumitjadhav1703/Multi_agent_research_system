# Assets Directory

This directory is intended to store screenshots and other visual assets for the `ResearchMind` README.

Due to the application requiring active API keys (Mistral AI and Tavily Search) to run the full pipeline, screenshots could not be automatically generated during setup.

## Required Screenshots

To complete the README, please run the application locally and capture the following screenshots. Save them with the exact filenames listed below so the README.md placeholders render them correctly:

1. **`home_page_placeholder.png`**: A screenshot of the Streamlit app immediately after running `streamlit run app.py`, showing the clean UI and input field.
2. **`pipeline_running_placeholder.png`**: A screenshot captured while the pipeline is executing (e.g., when the spinners or progress UI are active).
3. **`final_report_placeholder.png`**: A screenshot of the completed markdown report rendered in the Streamlit UI.
4. **`critic_feedback_placeholder.png`**: A screenshot showing the Critic Agent's feedback and scoring panel in the UI.

Once these images are added, the main `README.md` will automatically display them in the **Screenshots** section.
